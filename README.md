# Hackintosh-MAXSUN--H610M挑战者-12100F-rx5700

    如果帮助到你，可以给个star，谢谢🙏
## 配置
- 铭瑄H610m挑战者
- 12100F
- Radeon RX 5700
- 8*2 3200hz
- 声卡 ALC897


#### 支持
    Monterey 12.7.2 ,也兼容12.0~12.7.1版本，和Ventrua 13~12.6版本，可能有些小版本不识别声卡，独显驱动
#### 不足：
    不支持蓝牙、wifi，需要可以买博通免驱卡
